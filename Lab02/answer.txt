Question:
- If you create a constructor method to build a DVD by title then create a constructor method to build a DVD by category. Does JAVA allow you to do this?
- Có, Java cho phép bạn tạo nhiều constructor trong cùng một lớp, điều này được gọi là nạp chồng constructor (constructor overloading). Bạn có thể tạo các constructor khác nhau với các tham số khác nhau để xây dựng các đối tượng DVD theo tiêu đề hoặc theo thể loại.
